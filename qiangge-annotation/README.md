# 昆曲腔格标注研究

本目录承载与昆曲腔格识别、分析和标注有关的独立子项目。各子项目可以向后续腔格研究提供特征或中间结果，但不强制共享同一种算法、模型或服务接口。

## 子项目

- `tone-yinyang/`：依据《韵学骊珠》四声阴阳体系，为标注系统内建逐字轨生成四声草稿。该子项目目前使用 DeepSeek 辅助推断，但它本身是独立工具，不代表未来腔格任务必须使用大模型。

## 规划原则

- 四声、字音、工尺、板眼、声学和腔格结果保持来源可追溯。
- 机器结果必须与人工核定结果区分，不把模型推断冒充原始文献结论。
- 后续腔格任务可独立选择规则系统、声学模型、统计方法或其他模型。
- 跨子项目只共享经过明确约定的数据字段，不共享偶然形成的运行时实现。
